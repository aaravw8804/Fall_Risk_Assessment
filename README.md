# Fall_Risk_Assessment

📊 Fall Risk Assessment Using Machine Learning
This project leverages patient incident data from hospitals (2012–2017) to predict the likelihood of fall-related risks using a Random Forest Classifier.
The dataset undergoes thorough preprocessing, including missing value handling, categorical encoding, and age range transformation. 
Key features such as shift timings, department, severity, and patient demographics are analyzed to classify the risk level (High vs. Low/Moderate).

The system enables healthcare professionals to identify high-risk cases proactively, enhancing hospital safety protocols. 
Evaluation metrics like accuracy, precision, recall, F1-score, and confusion matrix are used to validate model performance, achieving an accuracy of over 85%.
